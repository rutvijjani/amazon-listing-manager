Instructions
CSV Format Requirements:
  • First row must contain column headers
  • SKU column is required for all updates
  • For ASIN mode, include both asin and sku columns
  • Use UTF-8 encoding for special characters
  • Maximum 10,000 rows per file
Column Descriptions:
sku
Your seller SKU (required for every update)
asin
Amazon ASIN (optional in SKU mode, required in ASIN mode)
price
Product price (for price updates)
currency
Currency code: INR, USD, EUR, etc.
quantity
Available inventory quantity
fulfillment_channel
DEFAULT (FBM) or AMAZON (FBA)
