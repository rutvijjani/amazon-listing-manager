Setup Instructions
Prerequisites:
  • Active Amazon Seller Central account
  • Professional selling plan (for API access)
  • Amazon SP-API application registered (for production)
Required Credentials:

This application needs the following environment variables configured:

  • LWA_CLIENT_ID - Login with Amazon Client ID
  • LWA_CLIENT_SECRET - Login with Amazon Client Secret
  • AWS_ACCESS_KEY_ID - AWS IAM User Access Key
  • AWS_SECRET_ACCESS_KEY - AWS IAM User Secret Key
  • SP_API_ROLE_ARN - IAM Role ARN for SP-API
For development/testing, update the .env file with your credentials.
Need Help?
Common Issues:
  • Invalid Seller ID: Check Seller Central > Settings > Account Info
  • Access Denied: Ensure your AWS IAM role has SP-API permissions
  • Token Expired: Tokens refresh automatically, but re-connect if issues persist
  • Product-type errors: Load the listing in Manual Update and follow the product-type guide before editing advanced attributes
Supported Marketplaces:

This app supports all Amazon marketplaces including India (Amazon.in), USA, UK, Canada, and many more.
